De onde vem o termo "código aberto"?

Mais de vinte anos atrás, o termo "software de código aberto" foi cunhado pela co-fundadora do Foresight Institute, Christine Peterson. Ela queria cunhar esse termo como uma forma de atrair recém-chegados a esse espaço.


O nome original seria "software livre", mas havia muita confusão sobre o que "livre" realmente significava. Para esclarecer a questão, Peterson propôs uma mudança de nome para "software de código aberto".
